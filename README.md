# josekipedia-parser

> Node v20.11.0  
> Typescript v5.6.2

Install dependencies:

```bash
yarn
```

Run script:

```bash
yarn download
```

With Docker:

```bash
make up logs
make stop
make down
make restart
make env
```

Create sgf:

```bash
yarn create-sgf
```

sgf will be stored in data/all-josekis.sgf
